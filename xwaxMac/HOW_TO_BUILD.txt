To build you will need the following:

- OS X 10.6
- Developer tools (including Xcode)
- libSDL (http://www.libsdl.org/release/SDL-1.2.14.dmg)
- libSDL_ttf (http://www.libsdl.org/projects/SDL_ttf/release/SDL_ttf-2.0.9.dmg

Building is only supported from Xcode (ie there is no command line build script)
